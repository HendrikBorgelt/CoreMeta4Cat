    CatCore Metadata Reference Model

    The CatCore describes the minimum information which must be reported with research data concerning the field of catalysis. This guideline helps to handle and standardize data based on the FAIR principle (Findable, Accessible, Interoperable, Reusable).

    URI: https://w3id.org/nfdi4cat/catcore

    Name: catcore-metadata
